# ClaimFlow

Minimal Next.js + Supabase starter for email/password authentication.

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Copy `.env.example` to `.env.local` and fill in your Supabase project values.

3. Run the app:

   ```bash
   npm run dev
   ```

## Included

- Login page
- Signup page
- Protected dashboard page
- Supabase SSR session middleware
- Feature-based auth structure
